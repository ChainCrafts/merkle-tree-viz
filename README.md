# Merkle Tree Visualization

A professional, interactive web application to demonstrate how Merkle Trees work. Built with React, TypeScript, and React Flow.

## Features

- **Interactive Tree Construction**: Enter transactions and watch the tree build dynamically.
- **Visual Learning**: Zoom, pan, and explore the tree structure with React Flow.
- **Proof Generation**: Select any leaf to generate a Merkle Proof and visualize the path.
- **Verification Simulation**: Verify proofs step-by-step with real-time hashing.
- **Responsive Design**: Works on desktop and tablet devices.
- **Dark Mode**: Fully themed UI.

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: TailwindCSS 4
- **State Management**: Zustand
- **Visualization**: React Flow
- **Crypto**: crypto-js (SHA-256)
- **Icons**: Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Run tests:
   ```bash
   npm test
   ```

4. Build for production:
   ```bash
   npm run build
   ```
